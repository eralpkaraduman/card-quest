# card-quest

react-native & react-native-web code sharing demo app

## Prerequisites
- `yarn install`
- `yarn link`
- `npx @react-native-community/cli doctor`

## Running web app
- `yarn web-start`
- `yarn ios` && `yarn android`

## Running react-native app
- cmd+m, debug settings, change host to 10.0.2.2:8081
